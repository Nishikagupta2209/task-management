
# Task Management Application

Full Stack MERN Application with JWT Authentication and Task CRUD.

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Environment Variables
See backend/.env.example

## Features
- Register / Login
- JWT Protected Routes
- Task CRUD
- Filtering & Sorting
- Indexed DB fields

## Deployment
Frontend: Vercel
Backend: Render / Railway
